Operator Precedence

First operators in this list bind stronger:

  1. ¬\neg¬ (Negation)
  2. ∧\wedge∧ (And)
  3. ⊕\oplus⊕ (Xor)
  4. ∨\vee∨ (Or)
  5. →\rightarrow→ (Implication)
  6. ↔\leftrightarrow↔ (Equivalence)

From this point on we will use multiplication (⋅\cdot⋅) instead of ∧\wedge∧ and +++ instead of ∨\vee∨. Also, instead of writing ¬a\neg a¬a we will write aˉ\bar{a}aˉ. Sometimes the equality sign (===) is used to denote syntactic equality, but here we will denote sematic equivalence with ===.
