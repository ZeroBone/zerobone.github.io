Числовой ряд

Пусть a1,a2,…,an,…a_1, a_2, \dots, a_n, \dotsa1​,a2​,…,an​,… - последовательность чисел.

Тогда сумма a1+a2+⋯+an+…a_1 + a_2 + \dots + a_n + \dotsa1​+a2​+⋯+an​+… называется числовым рядом, где a1,a2,…a_1, a_2, \dotsa1​,a2​,… называются членами ряда, а ana_nan​ - общий член ряда.

Ряд считается заданным, если определён общий член ряда как функция его номера nnn: an=f(n)a_n = f(n)an​=f(n).

Ряд обозначается:

∑n=1∞an=∑n=1∞f(n)\sum_{n=1}^{\infty}{a_n} = \sum_{n=1}^{\infty}{f(n)}n=1∑∞​an​=n=1∑∞​f(n)

Частичная сумма ряда

nnn-ой частичной суммой ряда называется сумма первых nnn членов ряда.

Построим последовательность частичных сумм ряда.

S1=a1S2=a1+a2S3=a1+a2+a3⋯Sn=a1+a2+a3+⋯+anS_1 = a_1 \\ S_2 = a_1 + a_2 \\ S_3 = a_1 + a_2 + a_3 \\ \cdots \\ S_n = a_1 + a_2 + a_3 + \dots + a_nS1​=a1​S2​=a1​+a2​S3​=a1​+a2​+a3​⋯Sn​=a1​+a2​+a3​+⋯+an​

Если существует конечный предел последовательности частичных сумм при n→∞n \to \inftyn→∞, то ряд ana_nan​ называется сходящимся.

Если предел бесконечный или не существует, ряд называется расходящимся.

Если ряд сходится, то он сходится к сумме ряда SSS:

∑n=1∞an=S\sum_{n = 1}^{\infty}{a_n} = Sn=1∑∞​an​=S

Гармонический ряд

Гармоническим называется ряд:

∑n=1∞1n=1+12+13+⋯+1n+…\sum_{n = 1}^{\infty}{\frac{1}{n}} = 1 + \frac{1}{2} + \frac{1}{3} + \dots + \frac{1}{n} + \dotsn=1∑∞​n1​=1+21​+31​+⋯+n1​+…

Этот ряд расходится.

Простейшие ряды

Единичный ряд

Рассмотрим ряд 1+1+1+…1 + 1 + 1 + \dots1+1+1+….

Построим последовательность частичных сумм:

S1=1S2=2S3=3…Sn=nS_1 = 1 \\ S_2 = 2 \\ S_3 = 3 \\ \dots \\ S_n = nS1​=1S2​=2S3​=3…Sn​=n lim⁡n→∞Sn=lim⁡n→∞n=∞\lim_{n \to \infty}{S_n} = \lim_{n \to \infty}{n} = \inftyn→∞lim​Sn​=n→∞lim​n=∞

Следовательно, ряд расходится.

Знакочередующийся единичный ряд

Рассмотрим ряд 1−1+1−1+⋯+1−…1 - 1 + 1 - 1 + \dots + 1 - \dots1−1+1−1+⋯+1−….

Построим последовательность частичных сумм:

S1=1S2=0S3=1S4=0…S_1 = 1 \\ S_2 = 0 \\ S_3 = 1 \\ S_4 = 0 \\ \dotsS1​=1S2​=0S3​=1S4​=0…

Предел SnS_nSn​ не существует, следовательно ряд расходится.

Геометрический ряд

Геометрический ряд представляет собой ряд, похожий на геометрическую прогрессию.

∑n=0∞aqn=a+aq+aq2+⋯+aqn+…\sum_{n = 0}^{\infty}{a q^n} = a + aq + aq^2 + \dots + aq^n + \dotsn=0∑∞​aqn=a+aq+aq2+⋯+aqn+…

Сходимость

∑n=0∞aqn\sum_{n = 0}^{\infty}{a q^n}n=0∑∞​aqn

Этот ряд сходится, если ∣q∣<1\vert q \vert < 1∣q∣<1.

Этот ряд расходится, если ∣q∣≥1\vert q \vert \ge 1∣q∣≥1.

Доказательство

Sn=b1(1−qn)1−q=a(1−qn)1−qS_n = \frac{b_1 (1 - q^n)}{1 - q} = \frac{a (1 - q^n)}{1 - q}Sn​=1−qb1​(1−qn)​=1−qa(1−qn)​

Рассмотрим 3 случая:

Пусть ∣q∣<1\vert q \vert < 1∣q∣<1. Тогда qn→0q ^ n \to 0qn→0 при n→∞n \to \inftyn→∞. Следовательно:

lim⁡n→∞a(1−qn)1−q=lim⁡n→∞a1−q\lim_{n \to \infty}{\frac{a (1 - q^n)}{1 - q}} = \lim_{n \to \infty}{\frac{a}{1 - q}}n→∞lim​1−qa(1−qn)​=n→∞lim​1−qa​

Получили конечное число, значит ряд сходится.

Пусть ∣q∣>1\vert q \vert > 1∣q∣>1. Тогда qn→∞q^n \to \inftyqn→∞ при n→∞n \to \inftyn→∞. Следовательно:

lim⁡n→∞a(1−qn)1−q=∞\lim_{n \to \infty}{\frac{a (1 - q^n)}{1 - q}} = \inftyn→∞lim​1−qa(1−qn)​=∞

Получили бесконечность, значит ряд расходится.

Пусть ∣q∣=1\vert q \vert = 1∣q∣=1. Тогда мы получаем ряд a+a+a+…a + a + a + \dotsa+a+a+….

Этот ряд расходится, так как предел последовательности частичных сумм равен ∞\infty∞ (см пример).

Свойства сходящихся рядов

Умножение на число

Если ряд умножить на число, не равное 000, его сходимость не изменится.

∑n=1∞kan=k∑n=1∞an\sum_{n = 1}^{\infty}{ka_n} = k\sum_{n = 1}^{\infty}{a_n}n=1∑∞​kan​=kn=1∑∞​an​

Сумма общих членов

Пусть ряд ana_nan​ сходится к сумме AAA и ряд bnb_nbn​ сходится к сумме BBB (AAA и BBB - конечные числа).

∑n=1∞an=A∑n=1∞bn=B\sum_{n = 1}^{\infty}{a_n} = A \\ \sum_{n = 1}^{\infty}{b_n} = Bn=1∑∞​an​=An=1∑∞​bn​=B

Тогда ряд, составленный из суммы общих членов сходится к A+BA + BA+B:

∑n=1∞(an+bn)=∑n=1∞an+∑n=1∞bn=A+B\sum_{n = 1}^{\infty}{(a_n + b_n)} = \sum_{n = 1}^{\infty}{a_n} + \sum_{n = 1}^{\infty}{b_n} = A + Bn=1∑∞​(an​+bn​)=n=1∑∞​an​+n=1∑∞​bn​=A+B

Принцип отбрасывания

Рассмотрим следующий ряд:

∑n=1∞an=a1+a2+a3+⋯+an+…\sum_{n = 1}^{\infty}{a_n} = a_1 + a_2 + a_3 + \dots + a_n + \dotsn=1∑∞​an​=a1​+a2​+a3​+⋯+an​+…

Мы можем отбросить определённое количество первых слагаемых ряда:

∑n=k+1∞an=ak+1+ak+2+⋯+an+…\sum_{n = k + 1}^{\infty}{a_n} = a_{k+1} + a_{k+2} + \dots + a_n + \dotsn=k+1∑∞​an​=ak+1​+ak+2​+⋯+an​+…

Получившийся ряд называется kkk-ым остатком ряда.

kkk-ый остаток ряда и сам ряд имеют одинаковую сходимость.

Доказательство

Пусть ρn\rho_nρn​ - nnn-ая частичная сумма остатка ряда.

ρn=ak+1+ak+2+⋯+ak+n=Sk+n−Sk\rho_n = a_{k+1} + a_{k+2} + \dots + a_{k+n} = S_{k+n} - S_kρn​=ak+1​+ak+2​+⋯+ak+n​=Sk+n​−Sk​

(Sk+nS_{k+n}Sk+n​ - частичная сумма исходного ряда).

По условию ряд ana_nan​ сходится, а значит предел частичной суммы исходного ряда равен конечному числу SSS:

lim⁡n→∞Sk+n=S\lim_{n \to \infty}{S_{k + n}} = Sn→∞lim​Sk+n​=S

Тогда:

lim⁡n→∞ρn=lim⁡n→∞(Sk+n−Sk)=S−Sk\lim_{n \to \infty}{\rho_n} = \lim_{n \to \infty}{(S_{k+n} - S_k)} = S - S_kn→∞lim​ρn​=n→∞lim​(Sk+n​−Sk​)=S−Sk​

SSS и SkS_kSk​ - конечные числа, значит ряд сходится.

Необходимое условие сходимости ряда

Если ряд сходится, то an→0a_n \to 0an​→0 при n→∞n \to \inftyn→∞, то есть lim⁡n→∞an=0\lim_{n \to \infty}{a_n} = 0limn→∞​an​=0.

Это не является достаточным условием сходимости ряда, то есть нельзя сказать, что если общий член стремится к нулю, то ряд сходится.

Достаточное условие расходимости ряда

Если общий член не стремится к 000, ряд расходится.

Пример

Рассмотрим следующий ряд:

∑n=0∞2n+32n+1\sum_{n = 0}^{\infty}{\frac{2n + 3}{2n + 1}}n=0∑∞​2n+12n+3​ lim⁡n→∞an=lim⁡n→∞2n+32n+1=[∞∞]=lim⁡n→∞22=1≠0\lim_{n \to \infty}{a_n} = \lim_{n \to \infty}{\frac{2n + 3}{2n + 1}} = \Big[\frac{\infty}{\infty} \Big] = \lim_{n \to \infty}{\frac{2}{2}} = 1 \ne 0n→∞lim​an​=n→∞lim​2n+12n+3​=[∞∞​]=n→∞lim​22​=1​=0

Следовательно, ряд расходится.
