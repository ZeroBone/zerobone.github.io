Интегральная сумма

Рассмотрим функцию f(x)f(x)f(x), которая непрерывна на [a,b][a, b][a,b].

Разобъём промежуток [a,b][a,b][a,b] на участки Δxi\Delta x_iΔxi​ произвольным образом.

Длина одного участка: Δxi=xi+1−xi\Delta x_i = x_{i+1} - x_iΔxi​=xi+1​−xi​

В каждом таком участке произвольным образом возьмём точку ξi\xi_iξi​, то есть ξi∈[xi,xi+1]\xi_i \in [x_i, x_{i+1}]ξi​∈[xi​,xi+1​].

Составим интегральную сумму.

Sn=∑i=0n−1f(ξi)ΔxiS_n = \sum_{i=0}^{n-1}{f(\xi_i) \Delta x_i}Sn​=i=0∑n−1​f(ξi​)Δxi​

При этом 0≤i≤n−10 \le i \le n-10≤i≤n−1, обозначим max⁡Δxi=λ\max{\Delta x_i} = \lambdamaxΔxi​=λ.

Определение

Если существует предел последовательности интегральных сумм SnS_nSn​, не зависящих от способа разбиения и не зависящих от выбора точек внутри частичных промежутков, то этот предел называется определённым интегралом и обозначается:

∫abf(x)dx=lim⁡n→∞,λ→0∑i=0n−1f(ξi)Δxi\int_{a}^{b}{f(x) dx} = \lim_{ n \to \infin, \lambda \to 0 } \sum_{i=0}^{n-1}{f(\xi_i) \Delta x_i}∫ab​f(x)dx=n→∞,λ→0lim​i=0∑n−1​f(ξi​)Δxi​

Геометрический смысл

Пусть f(x)≥0∀x∈[a,b]f(x) \ge 0 \forall x \in [a, b]f(x)≥0∀x∈[a,b]

Тогда интегральная сумма будет равна площади ступенчатой фигуры.

Sn=∑i=0n−1f(ξi)Δxi=Sступен.фигурыS_n = \sum_{i=0}^{n-1}{f(\xi_i) \Delta x_i} = S_{ступен. фигуры}Sn​=i=0∑n−1​f(ξi​)Δxi​=Sступен.фигуры​

При n→∞n \to \inftyn→∞ получится площадь криволинейной фигуры (криволинейной трапеции).

n→∞,λ→0⇒Sступен.фигуры→Sкрив.трапец.n \to \infty, \lambda \to 0 \Rightarrow S_{ступен. фигуры} \to S_{крив. трапец.}n→∞,λ→0⇒Sступен.фигуры​→Sкрив.трапец.​ ∫abf(x)dx=Sкрив.трапец.\int_{a}^{b}{f(x) dx} = S_{крив. трапец.}∫ab​f(x)dx=Sкрив.трапец.​

Таким образом, определённый интеграл равен площади криволинейной трапеции.
