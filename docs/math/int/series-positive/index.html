Положительный ряд

Ряд называется положительным, если an≥0a_n \ge 0an​≥0 для любого nnn.

Последовательность частичных сумм в таких рядах является монотонно возрастающей последовательностью.

Sn+1=Sn+an+1≥SnS_{n+1} = S_n + a_{n+1} \ge S_nSn+1​=Sn​+an+1​≥Sn​

Теорема о сходимости

Для того, чтобы положительный ряд сходился, необходимо и достаточно, чтобы последовательность частичных сумм была ограниченной сверху.

Доказательство

Необходимость

Пусть ряд сходится. Тогда существует предел последовательности частичных сумм, равный конечному числу SSS.

Если предел монотонно возрастающей последовательности равен конечному числу, эта последовательность ограничена сверху.

То есть SnS_nSn​ ограничена сверху, Sn≤SS_n \le SSn​≤S.

Достаточность

Пусть Sn≤SS_n \le SSn​≤S, SnS_nSn​ - монотонно возрастает.

Тогда, по теореме, существует предел последовательности частичных сумм, равный SSS, а значит, ряд сходится.

Признаки сходимости

Признак сравнения

Пусть ряды ∑n=1∞an\sum_{n=1}^{\infty}{a_n}∑n=1∞​an​ и ∑n=1∞bn\sum_{n=1}^{\infty}{b_n}∑n=1∞​bn​ положительные и начиная с некоторого n≥Nn \ge Nn≥N выполняется неравенство: an≤bna_n \le b_nan​≤bn​.

Тогда:

  1. Если ∑n=1∞bn\sum_{n=1}^{\infty}{b_n}∑n=1∞​bn​ сходится, то ∑n=1∞an\sum_{n=1}^{\infty}{a_n}∑n=1∞​an​ сходится.
  2. Если ∑n=1∞an\sum_{n=1}^{\infty}{a_n}∑n=1∞​an​ расходится, то ∑n=1∞bn\sum_{n=1}^{\infty}{b_n}∑n=1∞​bn​ расходится.

Доказательство

Обозначим nnn-ые суммы частичных рядов AnA_nAn​ и BnB_nBn​: An≤BnA_n \le B_nAn​≤Bn​

Пусть ∑n=1∞bn\sum_{n=1}^{\infty}{b_n}∑n=1∞​bn​ сходится. Тогда Bn≤BB_n \le BBn​≤B.

Следовательно, An≤Bn≤BA_n \le B_n \le BAn​≤Bn​≤B, то есть AnA_nAn​ ограничена.

Значит, ∑n=1∞an\sum_{n=1}^{\infty}{a_n}∑n=1∞​an​ сходится.

Пусть ∑n=1∞an\sum_{n=1}^{\infty}{a_n}∑n=1∞​an​ расходится.
