Точная верхняя и нижняя границы

Число MMM называется верхней границей множества AAA, если: ∀x∈A:x≤M\forall x \in A : x \le M∀x∈A:x≤M

Число MMM называется точной верхней границей множества AAA, если оно является наименьшей из верхних границ: sup⁡A=min⁡M\sup{A} = \min{M}supA=minM

Точная верхняя граница называется супремумом (супремум).

Число mmm называется нижней границей множества AAA, если: ∀x∈A:x≥m\forall x \in A : x \ge m∀x∈A:x≥m

Число mmm называется точной нижней границей множества AAA, если оно является наибольшей из нижних границ: inf⁡A=max⁡m\inf{A} = \max{m}infA=maxm

Точная верхняя граница называется инфимумом (инфимум).
