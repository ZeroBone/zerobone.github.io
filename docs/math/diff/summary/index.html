Функция

Пусть даны два множества: X и Y.

Функцией f называется любое правило, которое каждому элементу множества X ставит в соответствие некоторый элемент множества Y.

Точная верхняя и нижняя границы

Число M называется верхней границей множества A, если:

\forall x \in A : x \le M

Число M называется точной верхней границей множества A, если оно является наименьшей из верхних границ:

\sup{A} = \min{M}

Точная верхняя граница называется супремумом (супремум).

Число m называется нижней границей множества A, если:

\forall x \in A : x \ge m

Число m называется точной нижней границей множества A, если оно является наибольшей из нижних границ:

\inf{A} = \max{m}

Точная верхняя граница называется инфимумом (инфимум).

Предел функции по Коши

Число A называется пределом функции f(x) в точке x_0, то есть при x \to x_0 (x стремящемся к x_0), если для любого \varepsilon \ge 0 существует \delta_{\varepsilon} \ge 0, зависящая от \varepsilon такая, что для любого x \in X, такого что 0 \le |x - x_0| \le \delta_{\varepsilon}, выполняется неравенство |f(x) - A| \le \varepsilon.

\lim_{x \to x_0}{f(x)} = A \Leftrightarrow 
\forall \varepsilon > 0 \space \exists \delta_{\varepsilon} > 0 : \forall x \in X : 0 < |x - x_0| < \delta_{\varepsilon} \Rightarrow |f(x) - A| < \varepsilon

Предел функции по Гейне

Число A называется пределом функции f(x) в точке x_0, если для любой последовательности \{x_n\}_{n = 1}^{\infty}, такой что x_n \ne x_0 для любого n, и последовательность x_n стремится к x_0 при n \to \infty, следует, что и последовательность \{f(x_n)\}_{n=1}^{\infty} стремится к A при n \to \infty.

\lim_{x \to x_0}{f(x)} = A \Leftrightarrow \forall \{x_n\}, x_n \ne 0 : x_n \to x_0, n \to \infty \Rightarrow f(x_n) \to A

Предел последовательности

Число A называется пределом последовательности \{x_n\}_{n=1}^{\infty} если для любого \varepsilon > 0 существует число N, зависящее от \varepsilon, такое что для любого натурального числа n > N будет выполняться неравенство |x_n - A| < \varepsilon.

Бесконечно малые функции

Функция f(x) называется бесконечно малой при x \to x_0 по отношению к функции g(x), если:

\lim_{x \to x_0}{\frac{f(x)}{g(x)}} = 0

Функция f(x) называется бесконечно малой (то есть бесконечно малой по отношению к функции, тождественно равной единице) при x \to x_0, если:

\lim_{x \to x_0}{f(x)} = 0

Бесконечно большие функции

Функция f(x) называется бесконечно большой при x \to x_0 по отношению к функции g(x), если:

\lim_{x \to x_0}{\frac{f(x)}{g(x)}} = \infty

Эквивалентные функции

Функции f(x) и g(x) называются эквивалентными при x \to x_0, если:

\lim_{x \to x_0}{\frac{f(x)}{g(x)}} = 1 \Leftrightarrow f(x) \sim g(x), x \to x_0

Функции одного порядка

Функции f(x) и g(x) называются функциями одного порядка при x \to x_0, если:

\lim_{x \to x_0}{\frac{f(x)}{g(x)}} = M, M \ne 0, M \ne \infty

Непрерывная функция

Функция f(x) называется непрерывной в точке x_0, если для любого \varepsilon > 0 существует \delta_{\varepsilon} > 0, такая что для любого x, если выполняется условие |x - x_0| < \delta_{\varepsilon} то выполняется условие |f(x) - f(x_0)| < \varepsilon.

\forall \varepsilon > 0 \space \exists \delta_{\varepsilon} : \forall x, |x - x_0| < \delta_{\varepsilon} \Rightarrow |f(x) - f(x_0)| < \varepsilon

Фактически это определение предела, равного f(x_0).

Любая дифференцируемая функция является непрерывной во всех точках. Не всякая непрерывная функция является дифференцируемой во всех точках.

Функция f(x) называется непрерывной на множестве X, если она непрерывна в любой точке x_0 множества X.

Производная

Если следующий предел существует, то он называется производной f'(x).

f'(x) = \lim_{\Delta x \to 0}{\frac{\Delta y}{\Delta x}} = \lim_{\Delta x \to 0}{\frac{f(x + \Delta x) - f(x)}{\Delta x}}

Дифференциал

Пусть функция f(x) такова, что её приращение в точке x_0 может быть представлено в виде:

\Delta f(x_0) = A \cdot \Delta x + \beta(x_0, \Delta x)

A = A(x_0) - некое число.

Пусть также справедливо, что:

\lim_{\Delta x \to 0}{\frac{\beta(x_0, \Delta x)}{\Delta x}} = 0

Тогда функция f(x) называется дифференцируемой в точке x_0, выражение A \cdot \Delta x называется дифференциалом функции f(x) в точке x_0.

Дифференциал функции - это линейная часть приращения функции.

Касательная

Пусть на кривой выбраны точка M и точка M_1, а также проведена прямая (секущая) M M_1. Если перемещать точку M_1 вдоль по кривой, то прямая M M_1 будет вращаться вокруг точки M. Касательной к кривой в точке M называется предельное положение MT секущей M M_1, когда точка M_1 вдоль по кривой стремится к совпадению с M.

Уравнение касательной к графику функции f(x) в точке x_0:

y - y_0 = f'(x_0)(x - x_0)

где y_0 = f(x_0).
